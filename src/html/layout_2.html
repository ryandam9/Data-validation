Data Validation

This page presents data differences between Source & Target DBs.

  • To review column level differences, set DEBUG_DATA_VALIDATION to True and then view the log files @ logs folder.
  • The records validated in source and target DBs are available at data_validation folder.
  • Total tables validated : total_tables_validated_count
  • Tables completely matched : complete_match_tables_count
  • Tables with differences : tables_with_differences_count
  • Tables skipped : skip_tables_count
  • Tables skipped due to errors : error_tables_count
